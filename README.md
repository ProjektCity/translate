## Translate Projekt City
Help us translate Project City so we can make it available to more users. It's easier than you might think!

### Translating via Crowdin

We use [Crowdin](https://translate.projektcity.com/) to manage translations. To contribute:

1. Visit our [Crowdin project page](https://translate.projektcity.com/).
2. Sign up or log in to your Crowdin account.
3. Select your language and start translating or reviewing existing translations.
4. Your contributions will be reviewed and included in future releases.

Thank you for helping make Project City accessible to everyone!
